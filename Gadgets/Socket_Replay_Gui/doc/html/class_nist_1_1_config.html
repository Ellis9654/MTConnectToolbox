Detailed Description

The Config class can be used to load simple key/value pairs from a file.

Note
An example of syntax: // An example of a comment username= Bob gender= male hair-color= black // inline comments are also allowed level= 42
An example of usage: Config config; config.load("myFile.txt");

std::string username = config["username"].str(); int level = config["level"].toNumber<int>();

Config config; config.load(inifile); OutputDebugString(config.GetSymbolValue("FANUC.INCHES", L"INCHES").c_str()); OutputDebugString(StrFormat("%f\n", config.GetSymbolValue("MAIN.MAXRPM", "9000").toNumber<double>()));
